All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The <abbr> element allows users to represent an abreviation or an acronym.

<article>

This <article> represents a section of a page.

<aside>

This <aside> represents a section in which a user can do a summary. It can be used as a footer.

<body>

The element <body> represent the main content of the document.

<br>

The <br> element allows users to make a line break.

<button>

The <button> element allos users to make a button click.

<code>

The <code> element represents a camputer code.

<dd>

The <dd> element is used with a description list.

<dl>

The <dl> element is known as a description list.

<div>

The <div> element is used for a division.

<dt>

The <dt> element is the term or name in a description list.

<DOCTYPE>

The <DOCTYPE> element is used to describe the type of a document.

<footer>

The <footer> element is the enclosing section of a document.

<h1-h1>

The <h1-h1> element represents the heading.

<h2-h2>

The <h2-h2> element represents the subheading.

<h3-h3>

The <h3-h3> element represents a subheading less iportant than h2.

<h4-h4>

The <h4-h4> element represents a subheadings less important than h3.

<h5-h5>

The <h5-h5> element represents a subheading less important than h4.

<h6-h6>

The <h6-h6> element represents the least important subheading.

<head>

The <head> element is the element which contain the metadata for the document.

<header>

The <header> element is for the header section.

<iframe>

The <iframe> element allows user to make nested webpages.

<img>

The <img> element is used to show an image.

<li>

The <li> element allows user to make a list.

<nav>

The <nav> element represents a navigtion for a document.

<ol>

The <ol> element is used for making a ordered list.

<p>

The <p> element is for a paragraph.

<style>

The <style> element is used to make a style for a document.

<summary>

The <summary> element allows author to make a summary.

<table>

The <table> element is used to create a table.

<tbody>

The <tbody> element is used with a table to make rows.

<td>

The <td> element represens the data cell in a table.

<tfoot>

The <tfoot> element is for the row summary in a table.

<th>

The <th> element represents the header cell in a table.

<title>

The <title> element is for document title or name.

<tr>

The <tr> element is for the row of cell in a table.

<ul>

The <ul> element is for making a unordered list.
